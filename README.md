# Блок автоматизированного тестирования

## Тест кейсы

| ID   | Наименование                                         | Предусловия                | Входные данные                              | Ожидаемый результат                                                                              |
|------|------------------------------------------------------|----------------------------|---------------------------------------------|--------------------------------------------------------------------------------------------------|
| ТС-1 | Ввод данных в поля Name и Email                      | Открыта страница Profile   | Name = Ivan, Email = ivan@test.com          | Поля принимают значения, в блоке  pre отображаются введённые данные                              |
